# Startup-Monorepo-Template

This is a startup monorepo template, it consists of folders of services containing frontend, backend, and other service to be faster in launching new app 

Folder Structure 

```
# Project Folder Structure

## Root
Startup-Monorepo-Template/
├── README.md
├── .gitignore
├── frontend/
├── backend/
├── src/
│   ├── main.py
│   ├── utils.py
│   ├── config/
│   │   ├── config.yaml
│   │   └── logging.conf
│   └── modules/
│       ├── module1.py
│       └── module2.py
├── tests/
│   ├── test_main.py
│   └── test_utils.py
├── docs/
│   ├── index.md
│   └── api/
│       ├── overview.md
│       └── endpoints.md
```
